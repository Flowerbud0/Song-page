# 简易点歌台（带搜索）
纯前端极简点歌网页，仅展示歌曲列表+实时搜索功能，适配电脑/手机，可直接部署到GitHub Pages。

## 核心功能
- 清晰展示歌曲名+歌手名
- 实时搜索：支持歌曲/歌手模糊搜索，不区分大小写
- 自适应布局：手机、电脑打开都正常
- 无搜索结果时友好提示

## 如何修改歌曲列表
打开 `index.html` 文件，找到代码中 **【唯一需要你修改的地方】**，按 `歌曲名 - 歌手名` 格式替换即可，格式要求：
1. 每首歌单独一行，用双引号包裹
2. 除了最后一首，其他歌曲末尾加逗号
3. 支持合唱歌曲，格式如：`小酒窝 - 林俊杰/蔡卓妍`

## 如何使用
1. 本地使用：直接双击 `index.html` 文件，用浏览器打开即可
2. 在线使用：部署到GitHub Pages后，复制网址发给别人即可

## 部署说明
本仓库已适配GitHub Pages，按GitHub官方步骤开启即可，生成的网址可直接访问。
